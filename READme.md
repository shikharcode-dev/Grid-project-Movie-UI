# 🎬 Movie Collection - Assignment 4

This project is part of **Assignment 4**, where I created a **movie collection UI** using **CSS Grid Layout**.

The design is inspired by modern cinematic UI layouts, where multiple movie cards are arranged in a clean and responsive grid system.

---

## 🚀 Features

* Multi-section movie layout (Sci-Fi, Anime, Bollywood)
* Smooth hover effects with image/video transitions
* Cinematic dark theme UI
* Clean and structured grid system
* Responsive and visually engaging design

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* CSS Grid
* Flexbox (for internal alignment)
* Position (relative & absolute)

---

## 📚 What I Learned

* How to use `display: grid` properly
* Working with `grid-template-columns` & `grid-template-rows`
* Creating complex layouts using `grid-area`
* Combining Grid with Flexbox for better alignment
* Using position properties for overlays and effects
* Improving UI by observing real-world designs

---

## 🤖 Note

I used **AI tools (ChatGPT)** for guidance and learning concepts, especially for improving layout structure and UI ideas.
